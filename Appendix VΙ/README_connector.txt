To use the connector you need:

1) Download MT-5 platform https://www.metatrader5.com/en

2) Open the MQL5 editor (IDE button on the platform panel)

3) Copy the connector code from 
Appendix_VI_Connector_to_MetaTrader-5.mq5 
file into the editor

4) Create an executable Expert-file

5) Connect the Expert to the platform

6) For practical trading, you must select a broker (instructions here: https://www.metatrader5.com/en/trading-platform)

MQL5 documentation is here: https://www.mql5.com/en/docs
